Trireme Documentation

Getting Started

  • What is Trireme?
    • An AI-Automated Test Script Generator.
  • Main Features:
    • Mandatory Test Storage
    • SCR-based targeted test generation
    • Constructing full-flow tests with optimisation strategies
    • Excel Sheet conversion
  • Build your first project (Step-by-step guide)
    1. Create a project folder
    2. Create a module in the project folder
    3. Upload mandatory tests first; you may either do so manually or upload existing test cases in .xlsx format.

      4. Feel free to populate with more modules, and eventually test suites.
